# Gioco Indovina il Numero

## Descrizione
Gioco testuale sviluppato con l’aiuto di un’intelligenza artificiale.
L’utente deve indovinare un numero casuale generato dal programma.

## Obiettivo
Studiare un algoritmo semplice basato su cicli e condizioni.

## Linguaggio
- Python

## Funzionamento
Il programma genera un numero casuale tra 1 e 100.
L’utente inserisce tentativi finché non indovina.
Il programma fornisce suggerimenti.

## Complessità
- Tempo: O(n)
- Spazio: O(1)

## Uso dell’AI
L’AI è stata usata per:
- definire l’algoritmo
- scrivere il codice
- spiegare il funzionamento
